# Politics of Healthcare - Stakeholder Analysis (Synthetic Demo Dataset)

This dataset is designed for GradeGuard AI demo use.

## Scenario

- Course: `Politics of Healthcare`
- Assignment: `Stakeholder Analysis`
- Marking scale: `30 points`

## What is included

- `manifest.csv` with historical grading records
- `assignments/` with synthetic past submission texts

The examples intentionally vary in quality (low to top tier) and the awarded scores reflect those differences.

## Import steps

1. Open `/bias-checker` in the app.
2. Click `Upload Past Papers Folder`.
3. Select this folder:
   `demo-datasets/politics-healthcare-stakeholder-analysis/`
4. Grade your new submission in `/results` and apply professor-awarded score for fairness check.
